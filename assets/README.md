## Cómo Obtener Assets

Para obtener las imágenes reales:

1. **Miniaturas de Juegos**: Descarga de sitios como:
   - [ itch.io](https://itch.io) - Free game assets
   - [OpenGameArt.org](https://opengameart.org) - Art assets gratuitos
   - [ Freepik](https://freepik.com) - Imágenes premium (con atribución)

2. **Iconos y Logos**: 
   - [Font Awesome](https://fontawesome.com) - Íconos vectoriales
   - [Flaticon](https://flaticon.com) - Íconos premium
   - [Canva](https://canva.com) - Crear logos personalizados

3. **Avatar por defecto**: Usar cualquier imagen PNG de 128x128px

## Notas Importantes

- Todas las imágenes deben ser optimizadas para web (WebP, JPG)
- Tamaños recomendados:
  - Thumbnails: 300x200px (ratio 3:2)
  - Avatars: 128x128px (ratio 1:1)
  - Logo: 200x60px (transparente)

- Mantener el tamaño total por debajo de 500KB para carga rápida
